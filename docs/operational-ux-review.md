# Revisao UX operacional das telas criticas

## Escopo
- Leads.
- MasterMind CEO.
- Discovery.
- Clientes.

## Checklist
- Acoes nao podem truncar ou sair da tela.
- Filtros e grids devem funcionar em desktop e mobile.
- Densidade visual deve favorecer leitura operacional.
- Estados de aprovacao, aplicado, recusado e pendente devem ser claros.
- Cada tela deve preservar dark-first e hierarquia visual consistente.

## Aprendizado
Problemas de UX operacional reduzem velocidade de decisao, confianca e conversao. Toda revisao deve registrar antes/depois no MasterMind.
